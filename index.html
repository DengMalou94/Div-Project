HTML/CSS/JS


HTML (HyperText Markup Language)
It's used to structure and define the content of a web page. Function: It provides the basic building blocks of a webpage, including elements like headings, paragraphs, links, images, and forms. Syntax: HTML uses tags enclosed in angle brackets e.g., <div>, <a>, <p>. These tags are used to mark up the content and give it meaning.


CSS (Cascading Style Sheets)
It's used to style and layout web pages, controlling their appearance. Function: It separates the content of the webpage (written in HTML) from its design. CSS can modify colors, fonts, spacing, positioning, and responsiveness. Syntax: CSS uses selectors to target HTML elements and apply styles (e.g., p { color: blue; } styles all paragraph text to be blue).


JavaScript
It's is used to create dynamic and interactive elements on a web page. Function: It enables functionality such as form validation, interactive maps, animations, and event handling. JavaScript can manipulate HTML and CSS to update content and styles in response to user actions. Syntax: JavaScript code consists of variables, functions, and events (e.g., document.getElementById('myButton').addEventListener('click', function() { alert('Button clicked!'); });).

In summary, HTML structures content, CSS styles it, and JavaScript adds interactivity. Together, they form the core technologies for building modern web pages.
